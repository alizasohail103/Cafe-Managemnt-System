# ☕ Cafe Management System

A desktop-based Cafe Management & Billing System developed using **C# Windows Forms** and **SQL Server**.

---

## ✨ Features

- 🔐 Login System
- 📊 Dashboard
- 🍔 Menu Management
- 🛒 Order Management
- 🧾 Billing
- 📈 Reports
- 💾 SQL Server Database

---

## 🛠️ Technologies

- C#
- Windows Forms
- SQL Server
- Visual Studio
